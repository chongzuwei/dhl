# dhl system

In this project,we decided to solve the problem and challenges that are faced by dhl by designing an innovative digital platform called dhl System that aims to help Mandarin language learners to learn Mandarin from their tutors. With the growing global demand for Mandarin lessons, our system provides a personalized, engaging, and effective language learning experience that goes beyond traditional classroom methods.

## 🚀 Setup Instructions
Start the Vite development server

npm run dev

### Prerequisites
- **Node.js** (v18 or higher)
