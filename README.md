# dhl system

The focus of this project is the development of an Incident Reporting System designed to streamline DHL's response to a high volume of daily reports, including late deliveries, damaged parcels, and system errors. Currently, these reports are received through inconsistent channels such as email, WhatsApp, and handwritten warehouse instructions, which makes it difficult for management to identify real issues or prioritize resolution effectively. This initiative aims to solve these challenges by building an automated solution that handles the full lifecycle of an incident, from initial collection to final tracking.
